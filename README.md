# Tic Tac Toe ~ Minimal Black & White Edition

A clean, elegant, and fully responsive **Tic Tac Toe** game built with **React + Tailwind CSS**.  

![Tic Tac Toe Preview](https://github.com/Mehedi-Hasan-Himel/Tic-Tac-Toe-Game-with-react-js/blob/main/src/assets/websiteUI.png?raw=true)


## Features

- Ultra-minimalist black & white design
- Perfectly centered responsive layout
- Smooth hover & click animations
- Clear game status (Winner / Draw / Next player)
- Instant "New Game" reset
- Built with clean, reusable React components
- Mobile-friendly
